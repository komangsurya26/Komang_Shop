***Step to Start Project***

 1. Clone Project
 ```
 git clone https://github.com/komangsurya26/Komang_Shop.git
 ```
 2. 
 ```
 npm install
 ```
 3. 
 ```
npx sequelize-cli db:create
 ```
 4. 
 ```
npx sequelize-cli db:migrate
 ```
 5. 
 ```
npm start
 ```
